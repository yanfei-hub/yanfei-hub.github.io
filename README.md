# Yanfei Wang website update package

This package personalizes the Academic Pages template for Yanfei Wang.

## Files included

- `_config.yml`
- `_data/navigation.yml`
- `_pages/about.md`
- `_pages/research.md`
- `_pages/publications.md`
- `_pages/cv.md`
- `files/Yanfei_Wang_CV_2026_v2.docx`

## How to use

Upload these files into the root of the `yanfei-hub/yanfei-hub.github.io` repository and replace the existing files with the same names.

## Notes

- The Google Scholar link has been added.
- The CV page links to the uploaded DOCX file.
- You can later add a profile image by placing a file such as `profile.png` in the `images/` folder and setting `author.avatar` in `_config.yml`.
