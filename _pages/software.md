This page highlights research software, code repositories, and reproducible workflows related to my work in causal inference, target trial emulation, and longitudinal clinical modeling.

## Research software

### EHR-Target-Trail-Emulation

A code repository accompanying my work on operational target trial emulation using electronic health records. The project focuses on reconstructing treatment action from routine care data, decision-time alignment, support-aware estimation, and clinically interpretable strategy evaluation.

- **GitHub:** [yanfei-hub/EHR-Target-Trail-Emulation](https://github.com/yanfei-hub/EHR-Target-Trail-Emulation)
- **Related paper:** [npj Digital Medicine article](https://www.nature.com/articles/s41746-026-02563-z)
